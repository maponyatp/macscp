# Contributing to MacSCP

First off, thank you for considering contributing to MacSCP! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## How Can I Contribute?

### Reporting Bugs
*   Check the [Issues](https://github.com/tumishomaponya/macscp/issues) to see if the bug has already been reported.
*   If not, open a new issue. Include a clear title, a description of the problem, and steps to reproduce it.

### Suggesting Enhancements
*   Open a new issue and describe the feature you'd like to see, why it would be useful, and how it might work.

### Pull Requests
1.  Fork the repo and create your branch from `main`.
2.  If you've added code that should be tested, add tests.
3.  If you've changed APIs, update the documentation.
4.  Ensure the test suite passes.
5.  Make sure your code lints.
6.  Issue that pull request!

## Development Setup
1.  Clone the repository.
2.  Run `npm install` to install dependencies.
3.  Run `npm run dev` to start the development environment.

## Code of Conduct
Please be respectful and professional in all interactions. We aim to foster a welcoming environment for everyone.
